Poor
===

Poor is a WordPress starter theme based off _s that includes Gulp. 

The main modules:
* JSHint
* LiveReload
* Minify CSS
* Notify
* Plumber
* Rename
* SASS
* Uglify
* Watch

Usage
===

1. Rename the diretory to your custom theme name.
2. Open the directory in terminal
3. Run npm install
4. Run gulp
5. Make cool stuff
